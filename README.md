# Credit Card Fraud Detection

A machine learning project to detect fraudulent credit card transactions using classification techniques on highly imbalanced data.

## Overview
This project uses supervised learning to identify fraudulent transactions.  
It focuses on handling class imbalance and evaluating models using appropriate metrics.

## Technologies
- Python
- NumPy
- Pandas
- Scikit-learn

## Dataset
The dataset used is the Credit Card Fraud Detection dataset.

Source:
https://www.kaggle.com/mlg-ulb/creditcardfraud

> The dataset is not included in this repository to keep it lightweight.
